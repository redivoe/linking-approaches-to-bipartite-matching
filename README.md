# Linking the Comparison and Graphical Approaches to Bipartite Matching

This repository contains the reproducibility materials for the article Redivo, E. (2026+). *Linking the Comparison and Graphical Approaches to Bipartite Matching* (Under review).

It provides the code needed to reproduce the computational results and figures in the paper.  
To run the scripts, the R package [`linkbrl`](https://github.com/redivoe/linkbrl) is required.
